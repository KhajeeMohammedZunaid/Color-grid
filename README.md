# ColorGrid - Infinite Color Palettes

ColorGrid is a beautiful, infinite color palette generator and collection tool. Discover, generate, and save stunning color palettes for your next project.

## Features

- **Infinite Generation**: Scroll endlessly to discover new, procedurally generated color palettes.
- **Categories**: Filter palettes by mood and style (e.g., Pastel, Neon, Vintage, Monochromatic).
- **Favorites**: Save your favorite palettes to a dedicated collection.
- **Quick Copy**: Click any color to instantly copy its HEX code, or copy the entire palette array.
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices.
